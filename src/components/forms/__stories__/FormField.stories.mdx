# Form Fields

Form fields is an alternative of using [Input](?path=/story/docs-forms-input--page) directly. It's a custom Form element which can be used inside `Form` element and it represents a complete field with label, input and error messages.

## `TextField`

[TextField](?path=/story/components-forms-textfield--field) is an full custom element

<div className="example">
  <Form>
    <TextField label="Name" name="name" />
  </Form>
</div>

```js
<Form>
  <TextField label="Name" name="name" />
</Form>
```

### Validations
